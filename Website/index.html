Vera: Blockchain-Backed Trust

Securely issue and verify using blockchain technology.

GitHub Poster Report Vera

Features

  • Issue credentials stored immutably on chain using smart contracts.
  • Third-party verification without centralized authority.
  • Support for revocation and future expansion.

About Vera

Vera is a blockchain-native verification platform that enables universities to issue and third parties to verify academic credentials securely and transparently using Ethereum smart contracts. Its decentralized architecture reduces fraud and manual administrative effort while improving trust.

The system's design includes user authentication, credential issuance, verification workflows, and planned contract features such as revocation and batch verification.

System Architecture

The system connects the frontend app to Ethereum smart contracts deployed on a test network. Interactive pages support credential issuance and verification.
